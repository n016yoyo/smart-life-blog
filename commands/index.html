CLAUDE CODE 꿀팁

Claude Code 명령어 전체 + 실제 사용 예시

비개발자도 바로 쓰는 명령어 모음 · by @aitigermask
/resume
이전 대화 이어가기
예시 어제 만들던 작업, claude 켜고 /resume 으로 그 대화 그대로 이어서 진행.
/model
모델 바꾸기
예시 간단한 수정은 빠른 모델, 어려운 설계는 강한 모델로 바꿔 속도·비용 최적화.
/compact
대화 압축 (토큰 절약)
예시 대화가 길어 느려지면 /compact 로 핵심만 요약 압축. 맥락 유지하며 토큰 절약.
/clear
대화 초기화
예시 전혀 다른 작업 시작 전 /clear 로 이전 맥락 비워 엉뚱한 참조 방지.
/init
프로젝트 규칙(CLAUDE.md) 생성
예시 새 프로젝트에서 /init → 구조·컨벤션 분석해 규칙 문서 자동 생성. 이후 일관된 작업.
/agents
서브에이전트 (일 나눠 시키기)
예시 큰 작업을 여러 에이전트에 분배. 예: 검색 전담·리뷰 전담을 동시에.
/memory
기억(메모리) 편집
예시 "항상 한국어로, 엠대시 금지" 같은 영구 규칙 저장 → 매번 안 알려줘도 됨.
/mcp
외부 도구 연결
예시 피그마·노션·DB 등 외부 도구를 연결해 Claude가 직접 읽고 쓰게.
/review
코드 자동 검토
예시 변경한 코드 /review 로 버그·개선점 자동 점검.
/loop
반복 작업 자동 실행
예시 "5분마다 배포 상태 확인" 처럼 주기적 작업을 알아서 반복.
/cost
사용량·비용 확인
예시 이번 세션 토큰을 얼마나 썼는지 바로 확인.
/rewind
되돌리기(체크포인트)
예시 잘못된 변경을 이전 시점으로 롤백.
/config
설정 열기
예시 테마·권한·모델 기본값 등 변경.
Shift+Tab
플랜 모드
예시 바로 실행 말고 계획부터 세우게. 큰 작업 전에 안전하게.
