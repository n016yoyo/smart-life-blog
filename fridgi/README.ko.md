## Fridgi란?

Fridgi는 AI 기반 유통기한 관리 앱입니다. 장 보고 온 뒤 **영수증을 한 장 찍으면** AI가 식품명과 유통기한을 자동으로 등록합니다.

한국 가정은 평균적으로 구입한 식품의 30~40%를 유통기한 경과로 버리고 있습니다. Fridgi는 D-7, D-3, 당일에 푸시 알림을 보내 식재료를 제때 소비할 수 있도록 도와줍니다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| 📸 AI 영수증 스캔 | 마트 영수증 촬영 한 번으로 모든 식품 자동 등록 |
| 🔔 만료 알림 | D-7, D-3, 당일 푸시 알림 |
| 🧊 보관 구분 | 냉장실, 냉동실, 팬트리별 분류 관리 |
| 📊 낭비 대시보드 | 소비 vs 폐기 패턴 추적 |
| 📅 캘린더 뷰 | 이번 주 만료되는 식품을 한눈에 확인 |
| 🏠 홈 화면 위젯 | 앱을 열지 않고 만료 임박 식품 확인 |
| 🌍 4개국어 | 한국어, 영어, 일본어, 중국어 |
| 🔄 영수증 번역 | 한국 영수증을 사용자 언어로 자동 번역 |

## 사용 방법

1. **장 본 후 영수증 촬영**
2. **AI가 모든 식품을 자동 등록** (유통기한 포함)
3. **D-7, D-3, 당일 알림** 수신
4. **만료 임박 식품 위주로 식단 계획**
5. **낭비 대시보드에서 개선 추이 확인**
