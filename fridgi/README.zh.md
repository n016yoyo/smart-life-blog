## 什么是Fridgi？

Fridgi是一款基于AI的食品保质期管理应用。购物后只需**拍一张购物小票**，AI就能自动登记所有食品及其保质期。

普通家庭平均有30-40%的食品因过期而浪费。Fridgi在D-7、D-3和当天发送推送通知，帮助您及时消耗食材，减少浪费。

## 主要功能

| 功能 | 说明 |
|------|------|
| 📸 AI小票扫描 | 拍摄超市小票，一次性自动登记所有食品 |
| 🔔 过期提醒 | D-7、D-3、当天推送通知 |
| 🧊 存储分类 | 冷藏室、冷冻室、储物柜分类管理 |
| 📊 浪费仪表盘 | 追踪消耗与浪费趋势 |
| 📅 日历视图 | 一目了然查看本周到期食品 |

| 🌍 4国语言 | 韩语、英语、日语、中文 |
| 🔄 小票翻译 | 韩国小票自动翻译为您的语言 |

## 使用方法

1. **购物后拍摄小票**
2. **AI自动登记所有食品**（含保质期）
3. **D-7、D-3、当天收到提醒**
4. **优先使用即将到期的食材安排膳食**
5. **在浪费仪表盘查看改善情况**
