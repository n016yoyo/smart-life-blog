Google AI Essentials

Google · Coursera
무료 체험 후 유료 / 재정지원 시 무료

무엇을

Google이 만든 AI 활용 기본기 과정. 5개 코스(AI 소개 · 생산성 높이기 · 프롬프트 기술 · 책임감 있는 AI · 트렌드 따라잡기), 약 10시간 내외. 사전 지식 불필요.

비용

7일 무료 체험 후 월 $49 구독. 수료증을 받으려면 구독(유료) 또는 재정지원(financial aid) 승인이 필요해요. 7일 안에 끝내면 사실상 무료로 수료 가능.

따는 법

  1. 공식 페이지 접속 후 Enroll
  2. Coursera 가입 / 로그인
  3. 7일 무료 체험으로 시작하거나 "Financial aid available" 신청
  4. 5개 코스 수강
  5. 완료 시 Google 수료증 발급

받는 것

Google 수료증 (LinkedIn · 이력서 공유 가능)

공식: grow.google/ai-essentials · coursera.org

HubSpot Academy
100% 무료

무엇을

디지털 마케팅 자격증. 10개 레슨 · 영상 30개 · 약 3시간 24분. 콘텐츠 · 이메일 · SEO · 소셜미디어 · 인바운드 · 마케팅에 AI 활용까지.

완전 무료. 가입도 무료.

따는 법

  1. HubSpot Academy 무료 가입
  2. 강의 수강
  3. 시험 응시 (시간 제한, 한 번에 완료, 중간 이탈 시 12시간 잠금)
  4. 자동 채점 합격. 불합격 시 12시간 뒤 재응시(횟수 무제한)

받는 것

자격증 + 배지 (LinkedIn 공유). 유효기간 2년 (다시 수강 후 재합격하면 갱신).

공식: academy.hubspot.com/courses/digital-marketing

IBM
100% 무료

무엇을

GitHub Foundations

GitHub · 시험코드 GHF
$99 (학생은 무료)

무엇을

GitHub 기초와 협업 역량을 검증하는 입문 자격증. Git 기초 · 협업 · 저장소 작업. (보도 기준 60문항 · 120분 · 4지선다. 합격선은 GitHub가 비공개)

비용

응시료 $99 USD. ★ GitHub Education으로 인증된 학생은 무료 바우처(소진 시 종료). 학교 이메일로 학생 인증 후 신청.

따는 법

  1. 공식 등록 페이지(learn.github.com/certification/GHF)에서 등록
  2. Pearson VUE 방식 선택: 온라인 프록터드(감독) 또는 테스트센터
  3. 이름이 일치하는 정부발급 신분증 준비
  4. 예약(최대 90일 전) 후 응시

재응시: 1차는 24시간 후, 이후 14일 간격, 연 최대 5회.

받는 것

배지(Credly) + 증명서. 유효기간 최소 3년.

공식: docs.github.com · 학생 education.github.com

Claude Certified Architect

Anthropic · 2026.3.12 출시
가입·학습 무료 / 시험은 조건부 유료

무엇을

Anthropic이 낸 첫 공식 기술 자격증. Claude로 프로덕션 앱을 만드는 솔루션 아키텍트용 시험이에요. (보도 기준 60문항 · 120분 · 시나리오 기반. 다루는 영역: 에이전트 아키텍처 · Claude Code 워크플로우 · 프롬프트 엔지니어링 · 툴/MCP 통합 · 컨텍스트 관리)

비용 (정확히 알아두세요)

Claude Partner Network 가입은 무료(공식). 보도에 따르면 준비 코스(Anthropic Academy)는 무료, 얼리 액세스로 파트너사 직원 선착순 5,000명은 시험 무료, 그 이후 1회 $99예요.

주의: "100% 무료"는 사실이 아니에요. 무료 구간(가입·학습·선착순 5,000명)과 유료 구간($99)이 나뉘어요. $99·5,000명 수치는 Anthropic 공식 페이지엔 명시되지 않은 보도 정보라, 신청 전 공식 페이지로 꼭 확인하세요.

따는 법

  1. Anthropic Academy 무료 코스로 학습
  2. claude.com/partners에서 Claude Partner Network 무료 가입
  3. Skilljar 액세스 요청 → 바우처 수령 (보도 기준)
  4. 프록터링 시험 응시 (보도 기준)

받는 것

디지털 배지 (LinkedIn 공유) , 보도 기준

공식: anthropic.com/news/claude-partner-network · claude.com/partners · anthropic.skilljar.com
다시 한번: 가격·정책·시험 형식은 자주 바뀌어요. 특히 Claude 자격증의 가격·정원은 공식 확정이 아닌 보도 정보라, 신청 전 각 공식 페이지에서 최신 내용을 직접 확인하세요.
