HaruBrief (하루브리핑) — AI 아침 브리핑 앱

"하루가 시작되기 전에, 오늘을 파악하세요." 매일 아침 캘린더를 분석하고 AI가 오늘 일정을 한 문장으로 요약해드립니다.

HaruBrief란?

HaruBrief(하루브리핑)는 AI 기반 아침 브리핑 앱입니다. 매일 아침 캘린더를 분석해서 일정 요약, 점심 시간 여부, 퇴근 시간 예측, 번아웃 리스크까지 알려드립니다.

빽빽한 캘린더를 스크롤하며 하루를 파악할 필요 없습니다. HaruBrief를 열면 10초 안에 오늘이 어떤 날인지 알 수 있습니다.

주요 기능

기능 설명
🤖 AI 브리핑 일정을 자연어로 요약해주는 매일 아침 브리핑
🍱 점심 감지 오늘 점심 먹을 시간이 있는지 자동 확인
🏃 퇴근 예측 오늘 정시 퇴근이 가능한지 예측
🔥 번아웃 경고 일정이 과밀할 때 사전 경고
📊 인사이트 주간/월간 일정 패턴 분석
🔒 개인정보 보호 캘린더 데이터 기기 내 처리 / 익명화
🌏 다국어 한국어, 영어, 일본어, 중국어
🌙 다크 모드 다크 모드 완벽 지원

이렇게 작동합니다

  1. iPhone 캘린더 연동 (iOS 캘린더 / Google 캘린더)
  2. 매일 아침 브리핑 수신 (원하는 시간 설정 가능)
  3. 위험 일정 알림 수신 (과밀, 점심 누락 등)
  4. 인사이트 리포트로 주간/월간 일정 개선

이런 분께 추천합니다

사용자 활용 사례
회의가 많은 직장인 하루 회의 11개, 점심도 없는 날 사전 파악
매니저/팀장 팀 회의와 1:1 일정 과부하 감지
프리랜서 여러 프로젝트 일정을 한눈에 파악
번아웃 관리가 필요한 분 과밀 패턴 조기 발견
