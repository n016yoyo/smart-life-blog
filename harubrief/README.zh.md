## 什么是HaruBrief？

HaruBrief（하루브리핑）是一款基于AI的早间简报应用。每天早晨分析您的日历，提供日程摘要、午餐时间检测、下班时间预测和职业倦怠风险警告。

不再需要滚动满满的日历来了解今天的安排。打开HaruBrief，**10秒内了解今天是什么样的一天**。

---

## 主要功能

| 功能 | 说明 |
|------|------|
| 🤖 AI简报 | 用自然语言总结日程的每日早间简报 |
| 🍱 午餐检测 | 自动检查今天是否有时间吃午饭 |
| 🏃 下班预测 | 预测今天能否准时下班 |
| 🔥 倦怠预警 | 日程过密时提前预警 |
| 📊 洞察报告 | 周度/月度日程模式分析 |
| 🔒 隐私优先 | 日历数据在设备端处理/匿名化 |
| 🌏 多语言 | 韩语、英语、日语、中文 |
| 🌙 深色模式 | 完整支持深色模式 |

---

## 使用方法

1. **连接iPhone日历**（iOS日历 / Google日历）
2. **每天早晨收到简报**（可设置偏好时间）
3. **收到风险日程提醒**（过密、午餐缺失等）
4. **通过洞察报告**改善周度/月度日程安排
