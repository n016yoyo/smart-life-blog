## HaruBriefとは？

HaruBrief（ハルブリーフィング）は、AI搭載の朝のブリーフィングアプリです。毎朝カレンダーを分析し、スケジュール要約、ランチ時間の有無、退社時間予測、バーンアウトリスクまでお知らせします。

ぎっしり詰まったカレンダーをスクロールして一日を把握する必要はありません。HaruBriefを開けば**10秒で今日がどんな日か**わかります。

---

## 主な機能

| 機能 | 説明 |
|------|------|
| 🤖 AIブリーフィング | スケジュールを自然言語で要約する毎朝のブリーフィング |
| 🍱 ランチ検出 | 今日ランチを食べる時間があるか自動チェック |
| 🏃 退社予測 | 今日定時退社が可能か予測 |
| 🔥 バーンアウト警告 | スケジュールが過密な時に事前警告 |
| 📊 インサイト | 週間・月間スケジュールパターン分析 |
| 🔒 プライバシー重視 | カレンダーデータはデバイス内処理/匿名化 |
| 🌏 多言語 | 韓国語、英語、日本語、中国語 |
| 🌙 ダークモード | ダークモード完全対応 |

---

## 使い方

1. **iPhoneカレンダーと連携** (iOSカレンダー / Googleカレンダー)
2. **毎朝ブリーフィングを受信** (お好みの時間に設定可能)
3. **危険なスケジュールのアラート** を受信 (過密、ランチ欠如など)
4. **インサイトレポートで** 週間・月間スケジュールを改善
