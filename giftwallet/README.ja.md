## GiftWalletとは？

GiftWalletは、韓国のギフト券(기프티콘 / ギフティコン)を管理するAI搭載アプリです。KakaoTalkやSMSで届くギフト券を**写真1枚**で自動登録できます。AIがバーコード、金額、ブランド、有効期限を自動認識します。

韓国のギフト券は韓国語でしか表記されていないため、韓国語が読めない方には使いづらいものです。GiftWalletはギフト券情報を**日本語、英語、中国語**に自動翻訳して表示します。

## 主な機能

| 機能 | 説明 |
|------|------|
| 🤖 AI自動認識 | ギフト券の写真からバーコード、金額、ブランド、有効期限を即座に抽出 |
| 🌐 多言語対応 | 韓国語ギフト券を日本語/英語/中国語に自動翻訳 |
| 📲 ワンタップバーコード | 店頭でタップ1回でフルスクリーンバーコード表示 |
| ⏰ 有効期限リマインダー | カレンダー同期 + デバイス通知で自動アラート |
| 📊 ダッシュボード | カテゴリ別・ブランド別の資産概要 |
| 🎁 ギフトカード共有 | ギフト券をグリーティングカードに変えて送信 |
| 🏠 ホーム画面ウィジェット | アプリを開かずに期限切れ間近のカードを確認 (Android) |
| 📸 一括インポート | ギャラリーから100枚以上を一度に登録 |
