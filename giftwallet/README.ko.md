## GiftWallet이란?

GiftWallet(기프티지갑)은 AI 기반 기프티콘 관리 앱입니다. 카카오톡이나 문자로 받은 기프티콘을 **사진 한 장**으로 자동 등록합니다. AI가 바코드, 금액, 브랜드, 유효기한을 자동으로 인식합니다.

한국 기프티콘은 한글로만 표시되어 외국인이 사용하기 어렵습니다. GiftWallet은 기프티콘 정보를 **영어, 일본어, 중국어**로 자동 번역하여 외국인 친구나 동료에게도 편리합니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 🤖 AI 자동 인식 | 기프티콘 사진에서 바코드, 금액, 브랜드, 유효기한 자동 추출 |
| 🌐 다국어 지원 | 한국 기프티콘을 영어/일본어/중국어로 자동 번역 |
| 📲 원터치 바코드 | 매장에서 탭 한 번으로 전체 화면 바코드 표시 |
| ⏰ 만료 알림 | 캘린더 동기화 + 기기 알림으로 만료일 자동 알림 |
| 📊 대시보드 | 카테고리별, 브랜드별 기프티콘 자산 현황 |
| 🎁 선물 카드 | 기프티콘을 축하 카드로 꾸며서 전송 가능 |
| 🏠 홈 화면 위젯 | 앱을 열지 않고 만료 임박 기프티콘 확인 (Android) |
| 📸 일괄 등록 | 갤러리에서 100장 이상 한꺼번에 등록 |
