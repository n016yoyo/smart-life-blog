INTRO

이게 뭔데?

클로드코드 세션 여러 개 켜놓으면 어느 게 뭐 하는지 못 찾잖아요. 이 조합 하나면 세션들이 귀여운 캐릭터로 사무실에 서서 지금 뭐 하는지(생각 중인지, 다 끝났는지) 한눈에 보이고, 캐릭터를 클릭하면 그 세션이랑 브라우저에서 바로 이어서 얘기할 수 있습니다.

🏢 Pixel Agents

클로드코드 세션이 몇 개 떠 있는지, 뭘 하고 있는지 픽셀아트 사무실 화면으로 보여주는 도구. 포트 3100.

💬 CloudCLI

캐릭터를 클릭했을 때 그 세션을 이어서 대화할 수 있는 웹 기반 클로드코드 UI. 포트 3001.

둘 다 로그인 후 켜두면 재부팅해도 자동 실행되도록 등록합니다. 아래 프롬프트를 그대로 복붙하면 Claude Code가 설치부터 실행까지 다 해줍니다. 터미널 명령어를 몰라도 됩니다.
STEP 0 · 복붙

사전 준비물

새 컴퓨터에 Claude Code만 깔린 상태 기준입니다. 아래 프롬프트 하나로 Node 버전 확인부터 필요하면 설치까지 끝냅니다.

Claude Code에 붙여넣기
새 컴퓨터에 Claude Code만 있어. node --version으로 버전을 확인해줘. v20 이상이면 그냥 넘어가고, 없거나 낮으면 nvm을 설치해서 최신 LTS Node를 깔아줘.
STEP 1 · 복붙

Pixel Agents 설치

세션이 몇 개 떠 있는지, 뭘 하고 있는지 픽셀아트 사무실 화면으로 보여주는 도구예요.

Claude Code에 붙여넣기
~/apps 폴더에 https://github.com/pixel-agents-hq/pixel-agents 저장소를 클론해줘. 그 폴더에서 npm install, npm run build까지 실행하고, 끝나면 node dist/cli.js --port 3100 으로 백그라운드에서 실행해줘. 다 되면 http://localhost:3100 에서 픽셀아트 사무실 화면이 뜨는지 확인해줘.

설치 커맨드가 실제로 도는 화면 (LEOYANG 세션)

설치 끝나면 이렇게 보입니다 (세션마다 캐릭터, Thinking·Done 상태 표시)

Settings → Instant Detection(Hooks)도 켜주세요(브라우저에서 직접 클릭, 이것만 사람이 해야 해요). 이게 켜져야 이후 여는 모든 클로드코드 세션이 자동으로 캐릭터로 나타납니다.
STEP 2 · 복붙

CloudCLI 설치

세션을 클릭했을 때 이어서 대화할 수 있게 해주는 웹 UI(@cloudcli-ai/cloudcli 패키지)예요.

Claude Code에 붙여넣기
@cloudcli-ai/cloudcli 패키지를 npm으로 전역 설치해줘. 설치되면 cloudcli 명령으로 실행해줘. 콘솔에 뜨는 Server URL을 알려줘.

캐릭터 클릭하면 이 화면으로 넘어와서 바로 이어서 대화합니다

처음 브라우저로 열면 계정 생성 화면이 뜹니다(로컬 전용 single-user라 원하는 아이디·비밀번호로 만들면 돼요).

STEP 3 · 직접

Pixel Agents ↔ CloudCLI 연결

여기만 브라우저에서 사람이 직접 클릭해야 해요. http://localhost:3100 접속 → Settings 열기 → Remote Session Viewer URL에 아래 입력:

Remote Session Viewer URL
http://localhost:3001
저장되면 사무실 화면에서 캐릭터 클릭할 때마다 그 세션이 새 탭으로 열려서 바로 이어서 프롬프트 칠 수 있습니다.
OPTIONAL

다른 기기에서 접속하고 싶다면

같은 와이파이 아니어도 휴대폰·다른 PC에서 접속하려면 Tailscale 설치 후 로그인. 그러면 이 PC에 100.x.x.x 고정 IP가 생겨서 http://100.x.x.x:3100으로 접속 가능합니다.

CHECKLIST

확인 체크리스트

✓ localhost:3100 사무실 화면 ✓ Settings → Hooks 켜짐 ✓ localhost:3001 CloudCLI 로드 ✓ Remote Session Viewer URL 저장됨

지금 실행 중인 클로드코드 세션이 사무실 화면에 캐릭터로 보이는지 확인하면 끝.

TROUBLESHOOT

문제 생기면

증상확인할 것
캐릭터가 안 보임Settings의 Instant Detection(Hooks) 켜져 있는지 확인
포트 충돌3100·3001이 이미 쓰이는 중이면 다른 포트로 재실행(--port)
다른 기기 연결 안 됨Remote Session Viewer URL을 Tailscale IP로 바꿨는지 확인
